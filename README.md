# Useful Modpack 实用整合包

[![Modrinth](https://img.shields.io/badge/Modrinth-实用整合包-00AF5C?logo=modrinth)](https://modrinth.com/modpack/syzhb_monus)
[![License](https://img.shields.io/badge/License-GPL--3.0-blue)](https://www.gnu.org/licenses/gpl-3.0.txt)

一个轻量级的 Minecraft 1.20.1 Forge 整合包，专注于**性能优化**和**实用功能**，让你在保持原版风格的同时，获得更流畅、更便捷的游戏体验。

A lightweight Forge modpack for Minecraft 1.20.1, focused on **performance optimization** and **useful utilities**, delivering a smoother and more convenient vanilla+ experience.

---

## 🎯 特点 Features

- **⚡ 性能优化**: 通过 Embeddium + Oculus 大幅提升帧率，支持光影
- **🗺️ 便捷探索**: JourneyMap 实时小地图，Jade 方块信息显示
- **📦 物品管理**: JEI 合成表查询，MouseTweaks 快速整理
- **🔧 实用工具**: ConstructionWand 快速建造，AppleSkin 食物信息
- **🎨 界面美化**: CozyUI+ 让界面更舒适

## ⚙️ 系统要求 Requirements

- **Minecraft**: 1.20.1
- **模组加载器**: Forge
- **Java**: **Java 17** (Java 8 或 21 会导致崩溃)
- **内存**: 建议分配 **4GB** 以上

---

## 🚀 安装方式 Installation

### 使用启动器 (HMCL / PCL2 / Prism Launcher)
1. 下载 `.mrpack` 文件
2. 在启动器中导入整合包
3. 确保 Java 路径指向 Java 17
4. 启动游戏

### 手动安装
1. 下载整合包文件
2. 解压到 `.minecraft` 目录
3. 安装对应版本的 Forge
4. 将模组文件放入 `mods` 文件夹

---

## 📝 注意事项 Notes

- **必须使用 Java 17**，否则游戏会崩溃
- 如需使用光影，请在视频设置中启用 Oculus
- 建议关闭不必要的后台程序以释放内存
